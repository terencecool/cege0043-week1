User guide of Question Setting App

This user guide will help user to operate the following App more easily and efficiently during the whole process by step-bystep instructions with screenshots. If you have further concern or technical problems, please contact App developer to get more detail information.


  • Before launch Apps

    • Make sure the phone connects to internet (phone signal, WLAN, or Wi-fi)

    Note: The setting pages could be different in other smartphones, the example above just as a reference


  • Install Question Setting App to your phone

    • Download “Question Setting App” and install




  • Set up for Question Setting App

    • Turn on “location” tracking functionality on settings menu



  • Operation instruction

    • Sep 1: Launch Question Setting App by clicking icon on the screen

    Sep 2: Pick a point by clicking Leaflet map, the coordinate will automatically detect and pop up

    Sep 3: Type in question and answers (including the “number” of correct answer) you want in the blank box

    Sep 4: Submit your question by clicking “Upload Question” and check the correctness of question from popup alter message

    Sep 5: If you want to create more questions, please repeat the workflow from step 2
